# 🛍️ Customer Shopping Behavior Analysis

A comprehensive end-to-end data analytics project that analyzes customer shopping behavior using **Python**, **PostgreSQL**, and **Power BI**. This project demonstrates the complete data analytics workflow, from data cleaning and preprocessing to SQL-based business analysis and interactive dashboard development.

---

## 📌 Project Overview

Retail businesses generate massive amounts of transactional data every day. Analyzing this data helps organizations understand customer behavior, identify purchasing trends, optimize marketing strategies, and improve overall business performance.

In this project, a customer shopping dataset containing **3,900 transactions** was analyzed to uncover meaningful business insights related to customer demographics, spending habits, product performance, subscription behavior, and sales trends.

---

## 🎯 Project Objectives

The primary objectives of this project were to:

- Clean and preprocess raw customer transaction data.
- Perform exploratory data analysis (EDA) using Python.
- Store the cleaned dataset in PostgreSQL.
- Solve real-world business problems using SQL.
- Build an interactive Power BI dashboard.
- Generate actionable business recommendations.

# 💡 Key Insights

- Male customers generated significantly higher revenue than female customers.
- Clothing was the highest revenue-generating product category.
- Young Adults contributed the highest overall revenue.
- Loyal customers represented the largest customer segment.
- Express shipping customers had a slightly higher average purchase amount.
- Several products relied heavily on discounts to drive sales.
- Subscription status had little impact on average purchase amount but influenced total revenue due to customer volume.
